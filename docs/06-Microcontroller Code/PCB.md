## Overview

This code was created in MPLAB X IDE for the Curiosity Nano PIC18F57Q43 to demonstrate the subsystem functionality of the H-bridge. A button input to the microcontroller activates a sequence that drives the motor forward and backward. The code was generated without using MCC due to user-related difficulties in properly configuring the tool.
